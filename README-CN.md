# Manor Browser (庄园浏览器)

Manor Browser 是一款现代、高性能的 Android 网络浏览器，基于行业领先的 **GeckoView (Firefox)** 内核构建。它将 Firefox 的隐私性与兼容性，同优雅且高度可定制的用户界面相结合，致力于为您打造卓越的移动浏览体验。

## 🚀 核心功能

### 🛠️ 先进的浏览引擎
- **GeckoView 驱动**: 采用与 Firefox 相同的内核，确保顶级的网页标准兼容性、安全性和隐私保护。
- **流畅性能**: 针对速度和响应性进行了深度优化，轻松应对复杂的网页应用。

### 🎨 惊艳的视觉体验
- **动态背景**: 使用精美的动画效果装饰您的主页，包括：
  - 🌠 **流星雨 (Meteor Shower)**
  - 🌸 **樱花雨 (Sakura Rain)**
  - 🌧️ **雨天模式 (Rainy Day)**
  - ❄️ **雪天模式 (Snowy Day)**
  - 🌌 **极光之夜 (Aurora Night)**
  - 🎨 **纯色背景选项**
- **自适应 UI**: 完美支持日间/夜间模式，提供平滑的切换体验，并内置护眼主题（米色、绿色）。

### 🌍 全球多语言支持 (支持 20+ 语言)
全面支持 20 种主流语言，具备完美的排版渲染能力，特别包含对阿拉伯语等从右向左 (RTL) 语言的完整适配：
- 英语、简体中文、繁體中文、韩语、日语、俄语、法语、德语、西班牙语、意大利语、葡萄牙语、**阿拉伯语 (RTL)**、印地语、土耳其语、越南语、印尼语、波兰语、荷兰语等。

### 🛡️ 隐私与安全
- **密码管理器**: 安全存储并自动填充您的登录凭据。
- **Cookie 管理器**: 针对每个域名完整控制站点数据和 Cookie。
- **隐私优先**: 一键清除历史记录、Cookie 和缓存。
- **本地存储**: 您的所有个人数据（书签、历史记录、密码）均存储在设备本地，保障数据私有化。

### 🧰 强大的生产力工具
- **阅读模式**: 提供无干扰的阅读环境，支持自定义字体大小和背景主题。
- **自由复制模式**: 专门设计的模式，突破限制，可在任何网页上自由选择和复制文字。
- **扩展支持**: 支持安装插件和扩展功能，增强浏览器的可能性。
- **多标签管理**: 卡片式标签切换器，支持视觉预览和“一键关闭所有标签”。
- **高级下载器**: 灵活选择使用内置下载管理器或 Android 系统下载器。

### 📱 移动端深度优化
- **桌面/手机模式**: 快速在网站的移动版和电脑版之间进行切换。
- **搜索增强**: 支持 Google、百度等多种主流搜索引擎。
- **默认浏览器**: 可轻松将 Manor Browser 设置为系统的默认浏览器。
- **RTL 布局优化**: 针对阿拉伯语等语言进行了完美的镜像布局处理。

## 📦 安装说明

1. **克隆仓库**:
   ```bash
   git clone https://github.com/Olsc/ManorBrowser.git
   ```
2. **导入项目**: 使用 Android Studio 将项目作为 Gradle 项目导入。
3. **构建与运行**: 确保已安装 Android SDK，并在模拟器或物理设备上运行。

## 🛠️ 配置选项

进入“设置”菜单，您可以自定义：
- **常规**: 默认搜索引擎、界面语言及默认浏览器设置。
- **外观**: 背景动态效果、主题颜色以及阅读模式偏好。
- **隐私**: 管理已保存的密码、Cookie 和浏览历史。
- **下载**: 在内置下载器和系统下载器之间自由切换。
