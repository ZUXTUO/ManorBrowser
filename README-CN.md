# Manor Browser

Manor Browser 是一款基于强大 Firefox 引擎构建的现代、功能丰富的 Android 网络浏览器。它专注于隐私保护、性能表现和用户体验，提供简洁的界面和高级浏览功能。

## 功能特性

- **Firefox 引擎**: 基于可靠的 GeckoView (Firefox) 引擎，确保最佳的网页兼容性
- **标签页管理**: 支持多标签页浏览，具有视觉预览和便捷切换功能
- **注重隐私**: 设计时充分考虑隐私保护，保障您的浏览数据安全
- **自定义主题**: 支持浅色和深色模式，并配有动态背景效果
- **下载管理器**: 内置下载功能，支持文件和媒体下载
- **书签与历史记录**: 整理您喜爱的网站并访问浏览历史
- **密码管理**: 安全存储和自动填充登录凭据
- **高级搜索**: 支持多种搜索引擎（谷歌、百度等）
- **动态背景**: 精美的动画背景，包括流星雨、下雨、下雪、极光和樱花等效果
- **自由复制模式**: 增强的文字选择和复制功能
- **多语言支持**: 支持英语和中文

## 安装方法

1. 克隆仓库：
   ```bash
   git clone https://github.com/Olsc/ManorBrowser.git
   ```

2. 在 Android Studio 中打开项目

3. 构建并运行应用程序

## 使用方法

- **导航**: 使用返回、前进和刷新按钮进行标准导航操作
- **地址栏**: 点击中央栏位输入网址或搜索查询
- **菜单**: 访问设置、书签、历史记录、下载和其他功能
- **标签页切换**: 滑动切换标签页或使用标签页计数器管理多个页面
- **上下文菜单**: 长按链接、图片或文字以访问特定情境的操作选项

## 配置选项
