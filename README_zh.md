# Manor Browser (领域浏览器)

![Project Version](https://img.shields.io/badge/version-beta--0.0.1-blue)
![Engine](https://img.shields.io/badge/Engine-GeckoView--148-orange)
![License](https://img.shields.io/badge/License-Apache--2.0-green)

**Manor Browser** (领域浏览器) 是一款基于 Mozilla GeckoView (Firefox) 内核开发的现代化 Android 浏览器。

## 🌟 核心特色

### 1. 智能托管 & AI 助手
*   **Manor AI 助手**: 内置悬浮式 AI 助手，可直接阅读网页内容、回答问题并辅助用户操作。
*   **远程服务器通信**: 支持配置外部 AI 服务器地址，实现复杂的网页分析与自动化决策任务。

### 2. Firefox 同款强大内核
*   **GeckoView 驱动**: 继承火狐内核的高隐私性、强兼容性与卓越的渲染性能。
*   **插件扩展支持**: 支持安装与管理 Firefox 移动版扩展程序，打造属于你的定制化浏览器。

### 3. 极致视觉体验
*   **动态背景系统**: 内置流星雨、雨天、雪天、极光夜等多种精美的动态粒子背景，并支持自定义图片背景。
*   **现代交互**: 遵循 Material Design 设计规范，支持全屏模式、沉浸式浏览。
*   **高效导航**: 内置便捷侧边栏与**自定义底部栏按钮**，支持一键直达书签、插件或下载管理，极大地提升单手操作效率。

### 4. 强大实用工具集
*   **自由复制模式**: 突破部分网页的文字复制限制，随时随地自由提取文本。
*   **深度阅读模式**: 自动提取正文，支持自定义字号、护眼背景与配色。
*   **密码管理器**: 安全存储并自动填充账号密码，支持本地管理。
*   **Cookie 管理**: 独立的 Cookie 与网站数据管理面板，支持按域名查看与清理。

---

## 🛠️ 技术栈
*   **编程语言**: Java
*   **核心引擎**: Mozilla GeckoView
*   **编译版本**: Compile SDK 36, Min SDK 26 (Android 8.0+)
*   **核心组件**: OkHttp (网络通讯), Material Design (UI 框架)

## 🚀 快速开始

### 预构建版本
前往 [Releases](https://github.com/Olsc/ManorBrowser/releases) 页面下载最新的 APK 文件进行安装。

### 自行编译
1.  确保已安装 Android Studio (Jellyfish 或更高版本)。
2.  克隆仓库：
    ```bash
    git clone https://github.com/Olsc/ManorBrowser.git
    ```
3.  在 Android Studio 中打开项目。
4.  等待 Gradle 同步完成。
5.  点击 **Run** 按钮运行到模拟器或真机。

---

## 🔧 AI 与远程控制配置
在“设置 -> 智能托管”中配置 AI 服务器地址。开启后，浏览器将开始监听来自服务器的指令，您便可以通过内置的 AI 助手进行交互。
