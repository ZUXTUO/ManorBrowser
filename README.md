# Manor Browser

Manor Browser is a modern, feature-rich Android web browser built on the powerful Firefox engine. Designed for privacy, performance, and user experience, it offers a clean interface with advanced browsing capabilities.

## Features

- **Firefox Engine**: Built on the reliable GeckoView (Firefox) engine for optimal web compatibility
- **Tab Management**: Multi-tab browsing with visual previews and easy switching
- **Privacy Focused**: Designed with privacy in mind to protect your browsing data
- **Customizable Themes**: Light and dark mode support with dynamic background effects
- **Download Manager**: Built-in download functionality for files and media
- **Bookmarks & History**: Organize your favorite websites and access browsing history
- **Password Management**: Secure storage and auto-fill for login credentials
- **Advanced Search**: Support for multiple search engines (Google, Baidu, and more)
- **Dynamic Backgrounds**: Beautiful animated backgrounds including meteor showers, rain, snow, aurora, and sakura
- **Free Copy Mode**: Enhanced text selection and copying capabilities
- **Multi-language Support**: Available in English and Chinese

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Olsc/ManorBrowser.git
   ```

2. Open the project in Android Studio

3. Build and run the application

## Usage

- **Navigation**: Use the back, forward, and refresh buttons for standard navigation
- **Address Bar**: Tap the center bar to enter URLs or search queries
- **Menu**: Access settings, bookmarks, history, downloads, and other features
- **Tab Switching**: Swipe between tabs or use the tab counter to manage multiple pages
- **Context Menus**: Long-press on links, images, or text to access context-specific actions
